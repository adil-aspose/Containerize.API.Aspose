Produces the token lists for macros and for other things like and and .

Parameters
macroDefIf macro_def is true, the goal is to create the token list for a macro definition; otherwise the goal is to create the token list for some other TEX primitive: , , , , , , , , or .
xpandIf xpand is false, the token list will simply be copied from the input using get token.Otherwise all expandable tokens will be expanded until unexpandable tokens are left, except that the results of expanding '' are not expanded further.
Returns
A pointer to the tail of a new token list.
