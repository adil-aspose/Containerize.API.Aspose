◆ getRotationAngle()

def asposebarcode.Generation.BaseGenerationParameters.getRotationAngle (   self)

BarCode image rotation angle, measured in degree, e.g.

RotationAngle = 0 or RotationAngle = 360 means no rotation. If RotationAngle NOT equal to 90, 180, 270 or 0, it may increase the difficulty for the scanner to read the image. Default value: 0. This sample shows how to create and save a BarCode image. generator = BarcodeGenerator( EncodeTypes.DATA_MATRIX) generator.getParameters().setRotationAngle(7) generator.save("test.png")
