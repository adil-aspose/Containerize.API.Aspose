Detailed Description

Encapsulates a collection of IStyle objects.

[C++]
intrusive_ptr<Aspose::Cells::IWorkbook> workbook = Factory::CreateIWorkbook();
intrusive_ptr<IStyleCollection> styles = workbook->GetIStyles();
int styleIndex = styles->Add();
intrusive_ptr<IStyle> style = workbook->GetIStyles()->GetObjectByIndex(styleIndex);

NOTE: This class is now obsolete. Instead, please use CreateIStyle() in IWorkbook to create and manipulate style for workbook instead of Add() in IStyleCollection and use GetNamedIStyle() in IWorkbook to get named style instead of GetObjectByIndex(StringPtr).
