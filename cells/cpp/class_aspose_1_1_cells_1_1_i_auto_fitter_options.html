◆ GetAutoFitMergedCells()

virtual bool Aspose::Cells::IAutoFitterOptions::GetAutoFitMergedCells ( )
pure virtual

Indicates whether auto fit row height when the cells is merged in a row. The default value is false.

If the cells in merged in a row and auto fit the rows in MS Excel, MS Excel do not expand the row height. If this option is true, Aspose.Cells will expand the row height to fit the cells' data.
