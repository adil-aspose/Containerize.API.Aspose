Aspose::Words::Reporting Namespace Reference

Detailed Description

This namespace contains classes of the Aspose.Words' new reporting engine that supports report templates marked up with a language based on the LINQ Method Syntax.

This new reporting engine is more advanced than the original mail merge reporting engine MailMerging and allows to retrieve data from any .NET class, perform grouping, sorting, joining and interpret C# expressions.
