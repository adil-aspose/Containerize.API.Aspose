System::SharedPtr<IZoomFrame> zoomFrame = pres->get_Slides()->idx_get(0)->get_Shapes()->AddZoomFrame(150.0f, 20.0f, 50.0f, 50.0f, pres->get_Slides()->idx_get(1));
System::SharedPtr<IPPImage> image = pres->get_Images()->AddImage(System::Drawing::Image::FromFile(u"image.png"));

◆ get_ImageType()

virtual ZoomImageType Aspose::Slides::IZoomFrame::get_ImageType ( )
pure virtual

Gets the image type of a zoom object. Read ZoomImageType. Default value: Preview

Specifies whether the Zoom object is using the slide preview or a cover image.

Next example demonstrates changing Image Type to Preview value. In this case current image of a Zoom object changes to slide image:

System::SharedPtr<IZoomFrame> zoomFrame = pres->get_Slides()->idx_get(0)->get_Shapes()->AddZoomFrame(150.0f, 20.0f, 50.0f, 50.0f, pres->get_Slides()->idx_get(1), image);
zoomFrame->set_ImageType(Aspose::Slides::ZoomImageType::Preview);

Gets the duration of the transition between Zoom and slide. Read float. Default value: 1.0f

If not specified (TransitionDur = 0), it will use the destination slide transition and the timings associated with that transition.
