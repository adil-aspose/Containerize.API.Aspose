Detailed Description

Mathematical paragraph that is a container for mathematical blocks (IMathBlock)

Example:

auto shape = slide->get_Shapes()->AddMathShape(x, y, width, height);
auto mathParagraph = (System::DynamicCast_noexcept<Aspose::Slides::MathText::MathPortion>(shape->get_TextFrame()->get_Paragraphs()->idx_get(0)->get_Portions()->idx_get(0)))->get_MathParagraph();
