Detailed Description

UnauthorizedAccessException is thrown when access is denied by the operating system because of an I/O error or a security error. Never create instances of this class manually. Use the UnauthorizedAccessException class instead. Never wrap the UnauthorizedAccessException class instances into System::SmartPtr.
